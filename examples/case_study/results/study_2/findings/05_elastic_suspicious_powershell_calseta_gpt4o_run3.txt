## Investigation Summary

### Alert Classification and Severity Assessment
- **Alert Title**: Suspicious PowerShell Execution with Encoded Command
- **Severity**: High
- **Classification**: This alert is classified as a high-severity incident due to the use of encoded PowerShell commands, which are often associated with malware droppers and post-exploitation frameworks. The presence of a known command-and-control (C2) domain further elevates the risk.

2. **IP: 198.51.100.99**

### Overall Verdict
- **True Positive**: The alert is a true positive, as multiple indicators confirm malicious activity, including the use of a known C2 domain and a hash associated with a notorious malware family.

### Recommended Next Steps

1. **Immediate Actions**:
   - **Isolate the Host**: Use the endpoint agent to network-isolate the affected machine to prevent further communication with the C2 domain.
   - **Verify the Hash**: Confirm the hash's detection ratio and malware family classification on VirusTotal.

2. **Analysis**:
   - **Decode Encoded Commands**: Decode any PowerShell -EncodedCommand payloads to identify additional C2 domains or malicious activity.
   - **Check Network Connections**: Investigate outbound connections to the C2 domain/IP from any host in the environment.
   - **Scope the Incident**: Search for the same hash across all endpoints in the last 7 days to determine the extent of the compromise.

3. **Containment**:
   - **Block Indicators**: Add the malicious domains and IPs to the firewall block list to prevent further access.
   - **Quarantine the File**: Ensure the endpoint agent has quarantined the malicious file.
   - **Collect Forensic Artifacts**: Gather memory dumps, event logs, and prefetch files for further analysis.

4. **User Account Investigation**:
   - Investigate the activity of the user account associated with the SYSTEM account to determine if it was compromised.
