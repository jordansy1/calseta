## Investigation Summary

### Alert Classification and Severity Assessment
- **Title**: Suspicious PowerShell Execution with Encoded Command
- **Severity**: High
- **Status**: Open
- **Description**: The alert was triggered by the execution of a PowerShell command using encoded parameters, which is a common technique used by malware droppers and post-exploitation frameworks. The command attempted to download and execute a payload from a known command-and-control (C2) domain.

2. **IP: 198.51.100.99**

### Overall Verdict
- **True Positive**: The alert is a true positive, given the presence of multiple malicious indicators, including IPs, a domain, and a file hash associated with known malware activity.

### Recommended Next Steps

1. **Immediate Actions**:
   - Isolate the host using the endpoint agent to prevent further network communication.
   - Verify the hash on VirusTotal to confirm the malware family classification.
   - Identify the user account active on the host at the time of detection.

2. **Analysis**:
   - Decode the PowerShell encoded command to identify any additional C2 domains.
   - Check for outbound connections to the identified C2 domain/IP from any host in the environment.
   - Search for the same hash across all endpoints in the last 7 days to determine the scope of the incident.

3. **Containment**:
   - Block the identified malicious domains and IPs on the firewall.
   - Ensure the endpoint agent has quarantined the malicious file.
   - Collect forensic artifacts, including memory dumps, event logs, and prefetch files, for further analysis.

4. **Further Investigation**:
   - Investigate the pending verdicts for the URL and the user account to determine any additional risks or actions needed.
