## Investigation Summary

### Alert Classification and Severity Assessment
- **Title**: Suspicious PowerShell Execution with Encoded Command
- **Severity**: High
- **Status**: Open
- **Tags**: PowerShell, EncodedCommand, LivingOffTheLand, T1059.001, T1027, DefenseEvasion, Execution

The alert is classified as high severity due to the execution of a suspicious PowerShell command with an encoded payload, which is a common technique used for obfuscation and defense evasion. The presence of multiple malicious indicators further elevates the risk.

- **True Positive**: The alert is a true positive due to the presence of multiple confirmed malicious indicators, including a known C2 domain, malicious IPs, and a malware hash.

### Recommended Next Steps

1. **Immediate Actions**:
   - Isolate the host using the endpoint agent to prevent further network communication.
   - Verify the hash on VirusTotal to confirm the malware family and detection ratio.
   - Identify the user account active on the host at the time of detection.

2. **Analysis**:
   - Decode any encoded PowerShell commands to identify potential C2 domains.
   - Check for outbound connections to the identified C2 domain/IP from any host in the environment.
   - Scope the incident by searching for the same hash across all endpoints in the last 7 days.

3. **Containment**:
   - Block the identified malicious domains and IPs on the firewall.
   - Ensure the endpoint agent has quarantined the malicious file.
   - Collect forensic artifacts such as memory dumps, event logs, and prefetch files for further analysis.

4. **Further Investigation**:
   - Review the activity of the user account (Jorge Martinez) for any signs of compromise or misuse.
   - Conduct a thorough review of network logs to identify any additional suspicious activity related to the alert.
